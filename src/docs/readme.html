LocalLex README

离线极速英汉查词插件说明

功能

  • 完全离线英汉查词，不调用在线翻译、AI API 或远程脚本。
  • 双击网页英文单词，在文本附近显示查询界面。
  • 划选英文单词或短语，先显示 LocalLex 图标按钮，再点击显示释义。
  • 点击 Chrome 右上角 LocalLex 图标打开插件界面手动查词。
  • Alt+B 打开插件界面。
  • 支持 1-5 个英文 token 的短语、词形映射和基础规则还原。
  • 支持生词本、查询统计、CSV 导出、浅色/暗黑/跟随系统主题。

安装与使用

  1. 打开 chrome://extensions。
  2. 开启 Developer Mode。
  3. 点击 Load unpacked，选择 offline-ec-dict 目录。
  4. 如果插件已加载过，请点击 Reload，并刷新已经打开的网页。

使用时可双击单词、划选短语后点击 LocalLex 图标按钮，也可以点击插件图标或按 Alt+B 打开插件界面。

离线与隐私

LocalLex 只读取扩展包内的本地 JSON 文件，并将词库、生词本、设置和查询统计保存到本机 IndexedDB。用户选中文本和输入内容不会上传。

词库与例句来源

  • 主词库：ECDICT。
  • 真实英中例句：Tatoeba downloads。
  • Tatoeba per-language exports：exports/per_language。

例句只使用构建阶段导入的真实英中句对。没有真实例句时隐藏例句区，不使用模板句。

自定义词库格式

设置页支持导入 JSON，不直接导入 CSV。最简单格式是词条数组：

[
  {
    "word": "hello",
    "phonetic": "həˈləʊ",
    "translation": "int. 你好",
    "definition": "used as a greeting"
  }
]

完整格式：

{
  "version": "custom-1",
  "name": "My Dictionary",
  "entries": [
    {
      "word": "study",
      "phonetic": "ˈstʌdi",
      "translation": "n. 学习；研究；v. 学习；研究",
      "definition": "the activity of learning or gaining knowledge",
      "senses": [
        {
          "pos": "n.",
          "translation": "学习；研究",
          "definition": "the activity of learning or gaining knowledge"
        }
      ]
    }
  ],
  "forms": [
    {
      "form": "studies",
      "base": "study"
    }
  ],
  "examples": [
    {
      "key": "study",
      "examples": [
        {
          "en": "She studies English every morning.",
          "zh": "她每天早上学习英语。",
          "source": "custom"
        }
      ]
    }
  ]
}
